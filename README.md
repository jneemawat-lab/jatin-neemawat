# Inventory Management System (C)

A clean, efficient console-based Inventory Management System built using C. 

### Features
- **Static Array Allocation:** Uses fixed-size arrays for memory efficiency.
- **CRUD Operations:** Basic implementation of adding and displaying inventory items.
- **Data Structuring:** Uses `struct` to manage item properties effectively.

### How to Run
1. Clone this repo.
2. Compile using: `gcc inventory.c -o inventory`
3. Run: `./inventory`
